Lookbehind assertion is not fixed length

I am using grep for extracting string from file. If my regular expression looks like: 

grep -P -o -z '(?<=^ManagementCidr\s*=\s*).*'

I will get an error "Lookbehind assertion is not fixed length". Problem is in \s*. (?<=) works only if match if fixed size. In regular expressing for grep \K it will reset matched text at this point. So grep command that do what I want will be: 

grep -P -o -z '^ManagementCidr\s*=\s*\K(.*)
