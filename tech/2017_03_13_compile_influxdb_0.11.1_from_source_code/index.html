Compile influxdb 0.11.1 from source code

2017-03-13
database

Prepare essential directories and environment variables

export GOPATH=/src/influxdb/

mkdir -p $GOPATH/src/github.com/influxdata/

cd $GOPATH/src/github.com/influxdata/

export PATH=$GOPATH/bin:$PATH

Installing GDM

InfluxDB uses gdm to manage dependencies. Install it by running the following:

go get github.com/sparrc/gdm

Extract source code

tar zxf ~/Downloads/influxdb-0.11.1.tar.gz

ln -sf influxdb-0.11.1/ influxdb

Installing InfluxDB dependencies

cd influxdb

gdm restore

Build and install the binaries

Run the following command:

go clean ./...

go install ./...

and wait for a comment, the binaries will be installed to $GOPATH/bin:

ls -l $GOPATH/bin

Test the InfluxDB binaries just built

generate default influxdb configuration file and data directory

mkdir -p $GOPATH/etc

cd $GOPATH

bin/influxd config > etc/influxd.conf

mkdir -p $GOPATH/var/lib/influxdb

specific bind address and data directory

start the InfluxDB server

bin/influxd -config  etc/influxd.conf
