Clone git repository only some directories

2017/03/06

Categories: git Tags: git tips

  • First, do:
git clone --depth 1 [repo-root] [local_dir]
  • Then:
cd [local_dir]
  • And lastly:
git filter-branch --prune-empty --subdirectory-filter [repo-subdir] HEAD
