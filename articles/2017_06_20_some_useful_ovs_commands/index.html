Some useful OVS commands

2017/06/20

Categories: openvswitch Tags: openvswitch tips

OpenvSwitch Action
ovs-appctl fdb/show {bridge} Show MAC address table
ovs-appctl fdb/flush {bridge} Clear MAC address table
ovs-ofctl dump-ports-desc {bridge} Show port numbers
ovs-vsctl show Show OVS configuration
ovs-ofctl dump-flows {bridge} Show packet forwarding rules
ovsdb-tool show-log -m View command history
ovs-appctl ofproto/trace {bridge} in_port={ofport} Simulate packet flow
