OpenStack RESTful API example(Mitaka)

token-request.json file

{
    "auth": {
        "identity": {
            "methods": [
                "password"
            ],
            "password": {
                "user": {
                    "domain": {
                        "name": "Default"
                    },
                    "name": "admin",
                    "password": "openstack"
                }
            }
        },
        "scope": {
            "project": {
                "domain": {
                    "name": "Default"
                },
                "name": "admin"
            }
        }
    }
}

get the token

export TOKEN=`curl -si -d @token-request.json -H "Content-type: application/json" \
http://localhost:35357/v3/auth/tokens | awk '/X-Subject-Token/ {print $2}'`
