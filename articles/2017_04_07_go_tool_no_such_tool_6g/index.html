go tool: no such tool '6g'

2017/04/07

Categories: golang Tags: tips golang

  • Problem
davei@davei:/tmp> go tool 6g test.go
go tool: no such tool "6g"
  • Solution
6g were renamed to "compile" since Go 1.5. So use "compile" instead.
