Linux shared libraries

2016-04-02
clan

Shared libraries are collections of code that can be shared by programs. Unlike static (aka archive) libraries, they’re not part of the compiled program - they’re loaded in at the start of run-time if they’re not already being used. You don’t need to worry about them unless you’re a system manager or applications writer. Every shared library has … soname libname.so.version_number - the version number is incremented whenever the interface changes.
